# Module 12: Business Analytics – GreenGrocer Data Analysis
## Files Included
module12-assignment.py

## What I Practiced
In this assignment, I practiced applying the full analytics spectrum, including descriptive, diagnostic, and predictive analytics. I used Python libraries such as pandas, numpy, and matplotlib to analyze data, create visualizations, and build a basic regression model. I also developed skills in interpreting data, identifying patterns, and translating technical analysis into business insights and recommendations.

## Key Concepts
- Descriptive analytics (sales performance and trends)
- Diagnostic analytics (correlation and performance analysis)
- Predictive analytics (linear regression modeling)
- Data visualization using matplotlib
- Business decision-making using data

## Summary
This project demonstrates how data analysis can be used to evaluate business performance and support strategic decision-making. The results provide insights into sales trends, customer behavior, and operational efficiency, along with recommendations for improving overall performance.
